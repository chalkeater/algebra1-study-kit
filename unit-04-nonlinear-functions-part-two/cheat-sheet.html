The whole test in 10 sentences

  1. Absolute value = distance from 0 — the answer is always ≥ 0, never negative.
  2. Solve |expr| = c: isolate the bars, split into TWO equations (expr = c and expr = −c), solve both, check both in the ORIGINAL.
  3. |expr| = negative → no solution · |expr| = 0 → one solution · |expr| = positive → two solutions.
  4. AV inequalities: < or ≤ → AND, one connected interval ("between"). > or ≥ → OR, two separate rays ("split apart").
  5. Negative case of an inequality: flip the sign AND negate the number — then verify by plugging a value from inside AND outside the solution set into the ORIGINAL.
  6. g(x) = |x − h| + k: h positive slides RIGHT (the sign flips!), h negative slides LEFT; k positive slides UP, k negative slides DOWN.
  7. Increasing/decreasing intervals move with the vertex (h), not with k: decreasing on (−∞, h), increasing on (h, ∞).
  8. Adding/subtracting rational expressions (common denominator): combine numerators, but subtracting means distribute the negative to EVERY term in the second numerator.
  9. Radicals: simplify by pulling out perfect-square/cube factors; only add/subtract MATCHING radicands; multiply coefficient×coefficient and radicand×radicand, then simplify again.
  10. Dividing radicals: same-index roots combine into one radical (a/b); rationalize a monomial denominator by multiplying by that radical; rationalize a binomial denominator by multiplying by its CONJUGATE.

1Solving Absolute Value Equations

  • |x| strips the sign — |−7| = 7 and |7| = 7. It's a distance, never negative.
  • Isolate the bars first — undo everything outside them, exactly like any equation.
  • Split into two cases: positive (inside = value) and negative (inside = −value). Solve each like a normal linear equation.
  • Eyeball the isolated equation before splitting: positive constant → 2 solutions, 0 → 1 solution, negative constant → NO solution.
  • Always check both answers in the ORIGINAL equation — the bars will turn a negative result positive.

2Absolute Value Inequalities

  • < or ≤ → AND (one interval, "between two numbers"); > or ≥ → OR (two rays, "split apart").
  • Isolate the bars, write the positive case as-is, write the negative case with the sign FLIPPED and the number negated.
  • Open circle for strict (< >), closed circle for ≤ ≥.
  • Four ways to say the same set — x<−3 or x>7, (−∞,−3)∪(7,∞), {x | x<−3 or x>7}, and the number-line graph — practice converting between all four.
  • Verify by substitution: a number from INSIDE the solution set should make the original inequality true; a number from OUTSIDE should make it false.

3Absolute Value Functions & Translations

  • Parent f(x)=|x|: V-shape, vertex (0,0), domain = all reals, range y ≥ 0.
  • g(x) = |x − h| + k → vertex moves to (h, k). h positive → RIGHT (sign flips inside the bars!), h negative → LEFT. k positive → UP, k negative → DOWN.
  • Domain is always all reals. Range becomes y ≥ k.
  • Decreasing on (−∞, h), increasing on (h, ∞) — the break point is h, NOT k.
  • Writing g(x) from a graph: read the vertex (h, k) straight off, plug into |x − h| + k.

4Adding & Subtracting Rational Expressions

  • A rational expression is a fraction with variables — it behaves exactly like a numeric fraction.
  • Excluded values: set the denominator = 0 and solve — state those values, they're never allowed.
  • Same denominator? Combine straight into one numerator over that shared denominator.
  • Subtracting: distribute the negative sign to EVERY term in the second numerator before combining — the #1 mistake.
  • After combining: factor numerator and denominator, cancel any common factor, and re-state the excluded value.

5Simplifying, Adding & Multiplying Radicals

  • Simplify: factor out a perfect square/cube from the radicand, split the radical, take the root of the perfect part (works with variables too — √x⁷ = x³√x).
  • Add/subtract only matching radicands (same root, same stuff inside) — just like combining like terms.
  • Unlike radicands? Simplify each one first — a hidden match often appears.
  • Multiply: coefficient × coefficient, radicand × radicand — then simplify the result, don't stop early.
  • Distribute across a binomial like any other expression, then simplify each resulting term.

6Dividing Radicals

  • Quotient property: ⁿ√a / ⁿ√b = ⁿ√(a/b) — only when the indices match.
  • Rationalize a monomial denominator: multiply top & bottom by that same radical (or whatever completes a perfect root).
  • Rationalize a binomial denominator: multiply top & bottom by the conjugate (flip the middle sign) — square roots only.
  • Different indices? Convert both to rational exponents, find a common denominator on the exponents, combine, convert back to a radical.
  • Always fully simplify the final radical/fraction — check for leftover common factors.
